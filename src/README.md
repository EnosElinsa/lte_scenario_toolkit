# Reusable modules

This directory is the target location for the next migration phase. Planned modules are:

- `io.py`: vector, raster, CSV and manifest input/output;
- `spatial.py`: CRS conversion, geometry checks and point-in-boundary operations;
- `scenario.py`: candidate rectangle scanning, constraints and selection;
- `terrain.py`: DEM window reads and elevation sampling;
- `visualization.py`: 2D, 3D and publication figure rendering.

The first repository baseline keeps the existing root scripts unchanged.
